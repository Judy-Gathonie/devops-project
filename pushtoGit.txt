DevOps Project: From Scratch (Hands-On Guide)

We’ll build:

👉 A simple app → Dockerize → Push to GitHub → Automate with CI/CD → Deploy on Kubernetes

🧱 STEP 0: Prerequisites

Install these first:

Git
Docker
Minikube
kubectl
Node.js
Visual Studio Code

Check installs:

git --version
docker --version
node -v
kubectl version --client
📁 STEP 1: Create Project Folder
mkdir devops-project
cd devops-project

Initialize git:

git init
🌐 STEP 2: Build Simple Node.js App

Create file:

touch app.js

Paste this:

const http = require('http');

const server = http.createServer((req, res) => {
    if (req.url === '/hello') {
        res.writeHead(200, {'Content-Type': 'text/plain'});
        res.end('Hello DevOps World\n');
    } else {
        res.writeHead(404);
        res.end();
    }
});

server.listen(3000, () => {
    console.log('Server running on port 3000');
});

Create package.json:

npm init -y

Run locally:

node app.js

Open browser:
👉 http://localhost:3000/hello

🐳 STEP 3: Dockerize the App

Create Dockerfile:

FROM node:18
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 3000
CMD ["node", "app.js"]

Build Docker image:

docker build -t devops-app .

Run container:

docker run -p 3000:3000 devops-app

Test again:
👉 http://localhost:3000/hello

☁️ STEP 4: Push to GitHub

Go to GitHub and create a repo.

Then:

git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/devops-project.git
git push -u origin main
⚙️ STEP 5: Setup CI/CD Pipeline

Create folder:

mkdir -p .github/workflows

Create file:

touch .github/workflows/ci.yml

Paste:

name: CI Pipeline

on:
  push:
    branches: [ "main" ]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout code
      uses: actions/checkout@v3

    - name: Build Docker image
      run: docker build -t devops-app .

    - name: Run container
      run: docker run -d -p 3000:3000 devops-app

    - name: Test endpoint
      run: curl http://localhost:3000/hello

Push again:

git add .
git commit -m "Add CI pipeline"
git push

👉 Go to GitHub → Actions tab → see pipeline running 🎉

☸️ STEP 6: Setup Kubernetes (Minikube)

Start cluster:

minikube start
📦 STEP 7: Create Kubernetes Deployment

Create file:

touch deployment.yaml

Paste:

apiVersion: apps/v1
kind: Deployment
metadata:
  name: devops-app
spec:
  replicas: 2
  selector:
    matchLabels:
      app: devops-app
  template:
    metadata:
      labels:
        app: devops-app
    spec:
      containers:
      - name: devops-app
        image: devops-app:latest
        imagePullPolicy: Never
        ports:
        - containerPort: 3000
🌍 STEP 8: Create Service
touch service.yaml

Paste:

apiVersion: v1
kind: Service
metadata:
  name: devops-service
spec:
  type: NodePort
  selector:
    app: devops-app
  ports:
    - port: 80
      targetPort: 3000
      nodePort: 30007
▶️ STEP 9: Deploy to Kubernetes
kubectl apply -f deployment.yaml
kubectl apply -f service.yaml

Check:

kubectl get pods
kubectl get services
🌐 STEP 10: Access Your App
minikube service devops-service

Browser opens 🎉
